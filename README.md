# Vue版本Echarts实现中国地图三级钻取

## 1、运行及部署

### 1.1 本地运行
安装依赖

```
npm install
```
运行
```
npm run dev
```
访问：[localhost:8080](localhost:8080)
### 1.2 部署
打包
```
npm run build
```
